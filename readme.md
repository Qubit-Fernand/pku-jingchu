# PKU Jingchu

Notion-powered website for the PKU Jingchu Association.

This project is forked from
[transitive-bullshit/nextjs-notion-starter-kit](https://github.com/transitive-bullshit/nextjs-notion-starter-kit)
and follows the same structure as the local `pku-quantum` site.

## Configuration

Main site settings live in [`site.config.ts`](./site.config.ts).

- `rootNotionPageId`: `016135ae1ea14499a2464237daace5e1`
- `rootNotionSpaceId`: `e57fa69576c34239932f087196971d29`
- `domain`: `pku-jingchu.vercel.app`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
